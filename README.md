### 系统要求

- tvOS 14.0+
- Xcode 12.0+
- Swift 5.0+

### 安装方法

1. 克隆仓库
```bash
git clone https://github.com/yourusername/ATV-AiYiFan.git
```

2. 打开项目
```bash
cd ATV-AiYiFan
open ATV-AiYiFan.xcodeproj
```

3. 构建并运行项目
在 Xcode 中选择目标设备，点击运行按钮。
